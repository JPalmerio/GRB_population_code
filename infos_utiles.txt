Infos Utiles pour divers logiciels :

## Emacs ##
###########
ctrl + g : arrete une commande en cours
ctrl + y : colle (yank)
ctrl + s : search
esc + %  : remplace (then y or n or ! for all)
ctrl + space : select rectangle (then clic opposite corner)
ctrl + x r k : delete rectangle
ctrl + x r t : replace rectangle
ctrl + x r alt w : copy
ctrl + x r y : paste


## vpfit ##
###########
ctrl + h : efface (=backspace)
